# Laravel Marcho Ecommerce

This is my project Laravel Marcho Ecommerce

### Used

-   [Laravel] - php framework
-   Bootstrap
-   JQuery

### Installation

```sh
$ git clone
$ composer install
$ cp .env.example .env
$ php artisan key:generate
    setup DB in .env
$ php artisan migrate:fresh --seed
$ npm install
$ npm run dev
```
